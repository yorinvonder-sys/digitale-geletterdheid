## Nieuwe audit toevoegen

1. Voer de audit uit met het sjabloon in `../audit-plan-template-leerling.md`.
2. Sla het rapport op als `[missie-id]-audit.md` in deze map.
3. Voeg een rij toe aan de tabel hierboven met het correcte verdict:
   - **Klaar voor pilot** — score ≥ 80%, geen blockers
   - **Needs work** — score 60–79%, of 1+ medium blocker
   - **Niet inzetbaar** — score < 60%, of 1+ hard blocker
4. Werk de statistieken bij.
