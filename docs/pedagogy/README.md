## Kernkeuze: drie frameworks

```
4C/ID      →  Hoe is de opdracht opgebouwd? (taakontwerp)
Formatief  →  Wat leert de feedback de leerling? (feedbackkwaliteit)
SDT        →  Waarom gaat de leerling door? (motivatieontwerp)
Bloom      →  Op welk denkniveau werkt de leerling? (checklijst)
```

Zie `framework.md` voor de volledige uitleg en bronnen.

---

## Minimumnorm per missie

Een missie voldoet als:
- De taak **authentiek** is (4C/ID — V1)
- Leerlingen minstens **analyseren** (Bloom — V3)
- De feedback **meer zegt dan goed/fout** (Formatief — V6)
- Er een **tastbaar artefact** overblijft (Formatief — V7)

Zie `rubric.md` voor de volledige 7-punts checklijst.
