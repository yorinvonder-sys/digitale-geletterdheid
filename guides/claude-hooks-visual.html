2 Hoe werkt een PreToolUse hook?

De meest gebruikte hook: beslissen of een actie mag doorgaan of geblokkeerd wordt.

🤖 Claude wil een tool gebruiken
bijv. "npm test" of een bestand bewerken
⚡ PreToolUse HOOK draait
jouw script ontvangt de details
Mag dit
doorgaan?
✓ JA (exit 0)
✅ Tool draait normaal
PostToolUse hook volgt hierna
✗ NEE (exit 2)
🚫 GEBLOKKEERD
Claude ziet jouw reden en past aan

3 Wat stuurt Claude naar jouw script?

Claude stuurt JSON-data. Jouw script leest het, beslist, en geeft een exit code terug.

🤖

Claude Code

Stuurt details over wat hij wil doen

{
  "tool_name": "Bash",
  "tool_input": {
    "command": "rm -rf /"
  }
}
JSON via stdin ──────▶
◀────── exit 0 of exit 2
📜

Jouw Script

Leest JSON, neemt beslissing

#!/bin/bash
INPUT=$(cat)
CMD=$(echo "$INPUT" | jq -r
  '.tool_input.command')

if [[ "$CMD" == *"rm -rf"* ]]; then
  echo "Geblokkeerd!" >&2
  exit 2
fi
exit 0

4 De 4 soorten hooks

Kies het type dat past bij wat je wilt doen.

⚙️

Command

Draait een bash-script op je computer

Meest gebruikt! Format code, blokkeer acties, log alles
🌐

HTTP

Stuurt een POST naar een server

Audit logging naar extern systeem, team-brede regels
🧠

Prompt

Vraagt een AI-model om te beslissen

Slimme checks: "slagen de tests?" — AI beslist
🤖

Agent

Start een sub-agent die tools kan gebruiken

Complexe verificatie: bestanden lezen + commando's draaien

6 Praktijkvoorbeeld: .env beschermen

Stap voor stap: wat er gebeurt als Claude een beschermd bestand wil bewerken.

1
Claude wil .env bewerken
tool_name: "Edit", file_path: ".env"
2
PreToolUse hook vangt het op
matcher "Edit|Write" matcht → script draait
3
Script checkt het bestandspad
".env" zit in de beschermde lijst → BLOKKEREN
4
🚫 Exit 2 — actie geblokkeerd
"Blocked: .env bestanden zijn beschermd"
5
Claude past zich aan
Zoekt een andere oplossing die .env niet aanraakt

7 Alle hook-events op een rij

Elk event is een moment waarop jouw hook kan ingrijpen.

Event Wanneer Gebruik Fase
SessionStart Sessie begint of hervat Context injecteren Lifecycle
UserPromptSubmit Jij drukt Enter Input valideren Vóór
PreToolUse Vóór tool draait Blokkeren / toestaan Vóór
PostToolUse Ná tool succesvol Auto-format, logging Ná
PostToolUseFailure Ná tool faalt Error logging Ná
Stop Claude klaar met antwoord Check of taak af is Ná
Notification Claude wacht op jou Desktop notificatie Lifecycle
SessionEnd Sessie eindigt Cleanup, logging Lifecycle
