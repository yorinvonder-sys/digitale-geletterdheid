- **Content**: 8 gidsen, 3 clusterpagina's, 2 vergelijkingspagina's en 2 linkable assets live gezet.

## Geleerde Lessen
- Prerendering van dynamische gids-inhoud via `.md` bestanden werkt technisch goed voor metadata, maar volledige HTML-inhoud vereist een headless browser in de build-stap voor maximale SEO winst.
- Font-loading was een primaire bottleneck voor LCP; de preload-fix moet gemonitord worden via GSC Core Web Vitals rapportage.
