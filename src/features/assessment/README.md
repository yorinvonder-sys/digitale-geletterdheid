# Assessment

Deze folder bevat nulmeting, eindmeting, assessment tasks en assessment data.

Belangrijke ingangen:

- `AssessmentEngine.tsx`
- `types.ts`
- `data/assessmentRegistry.ts`
- `escaperoom/NulmetingFlow.tsx`
- `escaperoom/EindmetingFlow.tsx`

Assessmentwijzigingen raken leerlinggroei, docentrapportage en SLO-bewijsvoering. Houd data en task types lokaal bij deze feature en verifieer flows met `npm run doctor` plus een productie-build.
