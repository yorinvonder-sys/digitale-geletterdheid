# Games

Deze folder bevat game-overzichten, previews en losse game-implementaties.

Belangrijke ingangen:

- `GamesSection.tsx`
- `GameGallery.tsx`
- `GamePreview.tsx`
- `Bomberman/`
- `DrawingDuel/`
- `TypingTrainer/`

Games worden gebruikt vanuit de authenticated shell en AI Lab previews. Let bij wijzigingen op bundlegrootte, keyboard/touch-bediening en dat er geen gamecode naar publieke routes wordt gepreload.
